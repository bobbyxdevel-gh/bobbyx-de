# bobbyx-de
My personal website

## Setup

1. Install Ruby and Bundler (if not already installed):
   ```sh
   gem install bundler
   ```
2. Install dependencies:
   ```sh
   bundle install
   ```
3. Serve locally:
   ```sh
   bundle exec jekyll serve
   ```
   The site will be available at http://localhost:4000
